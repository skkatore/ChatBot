# ChatBot Helping Assistant 
This is a simple ChatBot web-application.

## Requirements
1. Node.js

## Installation
1. Clone the repository
```
git clone https://github.com/skkatore/ChatBot.git
```

2. Build the app
```
docker build -t chatbot .
```

3. Run the app
```
docker run -d -p 3000:3000 chatbot:latest
```
